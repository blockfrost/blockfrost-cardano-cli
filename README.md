# blockfrost-cardano-cli
Blockfrost Cardano CLI

## Features for initial release

* `cardano-cli query`
  *  `protocol-parameters`
  *  `tip`
  *  `stake-distribution`, prompt warning it will take long time
  *  `stake-address-info --address` 
  *  `utxo --address` 
* `cardano-cli transaction`
  *  `submit --tx-file`
